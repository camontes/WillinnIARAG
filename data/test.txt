LangChain es una librería para construir aplicaciones con modelos de lenguaje. Permite combinar documentos, bases de datos y LLMs para responder preguntas de manera inteligente.
.NET es una plataforma de desarrollo de software de código abierto y gratuita desarrollada por Microsoft, utilizada para crear una amplia gama de aplicaciones, desde aplicaciones web y de escritorio hasta aplicaciones móviles y de inteligencia artificial. React te ayuda a crear interfaces de usuario interactivas de forma sencilla.
React es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página. Es mantenido por Facebook y la comunidad de software libre. En el proyecto hay más de mil desarrolladores libres.
